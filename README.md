# Smart Device OS
A simplified cloud-based operating system for smart devices.

## Problem
Users can manage devices, customize menus, and run applications while data is stored centrally.

## Features
- User management
- Device and menu customization
- Application handling
- CLI interaction

## Tech Stack
- Java
- Spring Boot
- JPA (Hibernate)
- Maven

## Goal
Demonstrate object-oriented design, JPA modeling, and interactive CLI functionality.
